Fråga 1

Beskriv vad var och en av protokollen TCP och IP gör på sändande och mottagande sida.

Svar: TCP, transmission control protocol är ett protokoll som paketerar ner meddelanden i mindre paket genom nätverken som det ska färdas. På motsvarade mottagande sidan packas de paketerade meddelanden upp och sätts ihop för att sedan tas hand om vidare protokoll, upp i herarkin tex HTTTP. IP, internet protocol adresserar paketen som TCP delat upp/paketerat. På mottagande sida kan då varje paket verifieras med korrekt nummer.
