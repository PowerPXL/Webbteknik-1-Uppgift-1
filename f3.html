Fråga 3

F3-1. Vad är en regel i CSS och vad består den av?
Svar: En regel i CSS är exempelvis inställningar eller andra egenskaper som skrivs in, olika stilsättningar som används till html-dokumentet. CSS-regeler består av en selektorer som bestämmer vilket element som ska användas och deklatationer sätter egenskaperna inom det elementet.

F3-2. Vad är skillnaden mellan id och class? När bör man använda id respektive class?
Svar: Class är den större mängden som ska definieras och id är då ett specifikt element som behandlas. Man kan tänka sig ett klassrum med elever, där klassen är Class och eleverna id.
